# 바이브 코딩 — 사주 스터디 가이드

코딩을 처음 접하는 분들을 위한 바이브 코딩 설명 웹사이트입니다.

## 로컬에서 보기

1. 이 폴더에서 `index.html`을 더블클릭하거나
2. VS Code / Cursor의 Live Server 확장으로 실행

## Vercel로 공유하기 (추천)

1. [GitHub](https://github.com)에 새 저장소를 만듭니다
2. 이 폴더의 파일들을 push합니다
3. [vercel.com/new](https://vercel.com/new)에서 저장소를 연결합니다
4. **Deploy** 클릭 → `https://프로젝트이름.vercel.app` 링크가 생성됩니다
5. 카카오톡 등으로 링크를 공유하세요!

## 파일 구조

```
vibe_ppt/
├── index.html    # 메인 페이지
├── js/
│   └── app.js    # 인터랙션 (프롬프트 체험, 체크리스트 등)
└── vercel.json   # Vercel 배포 설정
```

## 스터디 진행 팁

- **1시간차**: 이 웹사이트를 함께 보며 "바이브 코딩이 뭔지" 이해
- **2시간차**: Cursor 설치 + 첫 프롬프트 실습 (위 체크리스트 따라하기)
- **3시간차**: 간단한 사주/운세 페이지를 Cursor로 직접 만들기
- **4시간차**: Vercel 배포 후 단톡방에 공유
