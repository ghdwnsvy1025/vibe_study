쉬운 비유

사주 공부할 때 옆에서 "이건 이렇게 보면 돼요" 알려주는 개인 선생님이에요. "이런 느낌의 홈페이지 만들어줘"라고 말하면, AI가 코드를 짜 줍니다.

설치 방법

  1. cursor.com 접속
  2. Download 클릭 → 설치 파일 실행
  3. 이메일로 가입 (무료로 시작 가능)
  4. 한국어로 대화하면 됩니다!

쉬운 비유

사주 프로그램이 돌아가려면 '기운'이 필요하죠? Node.js는 웹사이트를 만드는 도구들이 돌아가게 해주는 엔진이에요. 직접 코드를 안 써도, 설치만 해두면 AI가 알아서 씁니다.

설치 방법

  1. nodejs.org 접속
  2. LTS (안정 버전) 다운로드 — 초록색 버튼
  3. 설치 시 "Next"만 눌러도 OK
  4. 설치 확인: 터미널에 node -v 입력

쉬운 비유

사주 카페를 차렸는데 손님이 못 오면 의미가 없죠? Vercel은 만든 웹사이트에 주소(링크)를 붙여서, 카카오톡으로 공유할 수 있게 해 줍니다. 무료예요!

사용 방법

  1. vercel.com 가입 (GitHub 연동 추천)
  2. 프로젝트 폴더를 Vercel에 연결
  3. Deploy 버튼 클릭
  4. xxx.vercel.app 링크가 생김!
