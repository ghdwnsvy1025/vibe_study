바이브 코딩이란?

Vibe = 분위기, 느낌. 코딩을 몰라도 시작할 수 있는 새로운 방식

기존 코딩

  • 프로그래밍 언어를 직접 배워야 함
  • 문법 하나 틀리면 오류 발생
  • 시간이 오래 걸림

바이브 코딩 ✨

  • 한국어로 "이런 느낌으로" 말하면 됨
  • AI(Cursor)가 코드를 대신 작성
  • 빠르게 결과물을 볼 수 있음

Cursor AI

똑똑한 코딩 조수

  • 옆에서 "이렇게 하면 돼요" 알려주는 개인 비서
  • "이런 느낌의 홈페이지 만들어줘" → AI가 코드 작성
  • 한국어로 대화 가능, 무료로 시작
설치: cursor.com → Download → 가입 → 완료!
4 / 10
⚙️

Node.js

컴퓨터 안의 엔진

  • 웹사이트를 만드는 도구들이 돌아가게 해주는 엔진
  • 직접 코드를 안 써도, 설치만 해두면 AI가 알아서 사용
  • 한 번만 설치하면 계속 쓸 수 있어요
설치: nodejs.org → LTS (초록 버튼) → Next만 눌러도 OK
5 / 10
🌐

Vercel

웹사이트를 세상에 보여주는 창문

  • 만든 웹사이트에 주소(링크)를 붙여 줌
  • 카카오톡으로 링크 공유 → 누구나 접속 가능
  • 무료! GitHub와 연동하면 자동 배포
사용: vercel.com 가입 → GitHub 연결 → Deploy → xxx.vercel.app
6 / 10
