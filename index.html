바이브 코딩이란?

Vibe = 분위기·느낌. 코딩 몰라도 시작할 수 있어요

기존 코딩

  • 프로그래밍 언어를 직접 배워야 함
  • 문법 틀리면 오류 발생
  • 시간이 오래 걸림

바이브 코딩 ✨

  • 한국어로 "이런 느낌으로" 말하면 됨
  • AI(Cursor)가 코드를 대신 작성
  • 빠르게 결과물을 볼 수 있음

4단계만 기억하세요

느낌 정하기

2

AI에게 말하기

Cursor 채팅에 요청해요

3

결과 확인

만들어진 화면을 봐요

4

링크로 공유

🤖

Cursor AI

똑똑한 코딩 조수

  • 옆에서 알려주는 개인 비서 같아요
  • "홈페이지 만들어줘" → AI가 코드 작성
  • 한국어 대화 가능, 무료로 시작
설치
cursor.com → Download → 가입 → 끝!
⚙️

Node.js

컴퓨터 안의 엔진

  • 웹 도구가 돌아가게 해주는 프로그램
  • 직접 코드 안 써도, 설치만 하면 AI가 사용
  • 한 번만 설치하면 계속 씀
설치
nodejs.org → LTS (초록 버튼) → Next만 눌러도 OK
🌐

Vercel

웹사이트를 보여주는 창문

  • 만든 사이트에 주소(링크)를 붙여 줌
  • 카카오톡으로 공유 → 누구나 접속
  • 무료! GitHub 연동 시 자동 배포
사용
vercel.com 가입 → GitHub 연결 → Deploy

웹페이지 만드는 예시

모임 소개 페이지를 만든다고 가정해 볼게요

1

Cursor 실행

설치한 Cursor 프로그램을 엽니다

2

AI 채팅에 요청

아래처럼 말하면 됩니다

"모임 소개 웹페이지 만들어줘.
보라색 배경에 금색 포인트,
글씨 크게, 핸드폰에서 보기 좋게"
3

AI가 페이지 작성

1~2분 기다리면 코드가 만들어져요

예시 — 이어서

만들어진 뒤에 하는 일이에요

4

결과 확인

브라우저에서 화면이 잘 나오는지 봅니다

5

수정 요청 (선택)

"버튼 더 크게", "색 더 밝게" 처럼 말하면 AI가 고쳐 줍니다

6

Vercel로 공유

Deploy하면 xxx.vercel.app 링크가 생깁니다

💡 핵심

코드를 외울 필요 없이
말 → 확인 → 수정 → 공유
이 반복이 바이브 코딩이에요
